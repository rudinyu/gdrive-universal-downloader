# GDrive Universal Downloader

一段瀏覽器 Console 腳本，自動偵測 Google Drive 檔案類型並下載。

## 支援格式

| 類型 | 下載格式 |
|------|---------|
| PDF / Office 預覽 | .pdf |
| Google Docs | .docx |
| Google Sheets | .xlsx |
| Google Slides | .pptx |
| 圖片 | 原始格式 |
| 影片 | 來源 URL |
| 文字 / CSV | .txt |

## 使用方式

1. 在 Chrome / Edge 開啟 Google Drive 檔案
2. 按 `F12` 開啟 Console
3. 貼上 `index.html` 內的腳本並按 Enter

## 注意

本工具僅供用於您本人擁有或已獲授權存取的文件，請遵守著作權法規。
